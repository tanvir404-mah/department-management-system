Specialized Lab Divisions

💡

Digital Logic Lab

Dedicated to foundational hardware principles. Students work with microcontrollers, FPGA boards, breadboards, and logic gates to build and test digital circuits and embedded systems.

🌐

Advanced Networking Lab

Equipped with real-world Cisco and Juniper routers/switches. Focuses on network architecture, routing protocols, subnetting, VPN configuration, and cybersecurity practicals.

🧠

Software Lab I: Foundations

Core environment for mastering programming fundamentals. Focuses on procedural programming, basic algorithms, data structures, and debugging techniques using major IDEs.

⚙️

Software Lab II: Advanced Systems

Focuses on object-oriented programming (OOP), relational and non-relational databases (SQL, MongoDB), and introduction to enterprise application development frameworks (e.g., Spring/Django).

🏗️

Software Lab III: System Integration

The capstone environment for full-stack integration, DevOps practices, cloud deployment (AWS/Azure simulators), and collaborative project management using version control.

Hardware & Networking Assets

  • High-Fidelity Compute Nodes: 50x PCs (AMD Ryzen 7/Intel i7, 32GB DDR4 RAM, NVIDIA RTX 3060).
  • Advanced Networking Racks: Cisco Catalyst switches and ASA firewalls for dedicated cybersecurity and routing labs.
  • Prototyping & IoT Workshop: Dedicated area with soldering stations, oscilloscopes, and various ARM-based microcontrollers.
  • Virtual Reality Development Station: One dedicated high-end PC with Oculus Quest 2/Valve Index for immersive simulation development.

Governance Policy

Prohibited Activities

  • • Consumption of edibles or beverages near workstations.
  • • Modification of OS registries, security policies, or administrative passwords.
  • • Running unauthorized cryptocurrency mining or resource-intensive tasks.
  • • Any activity that violates academic integrity or institutional policies.

Mandatory Procedures

  • • Save all work to network storage before logging out or rebooting.
  • • Submit an incident report for any hardware failure or software anomaly.
  • • Maintain a clean and organized workspace for the next user.
  • • Adhere to assigned reservation times strictly.
